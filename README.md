# Kalkulator Fungsi

website kalkulator ini bisa membantu menyelesaikan perhitungan fungsi komposisi dan fungsi invers.
jika ingin menggunakan web ini, bisa dengan clone repository ini atau kunjugi [kalkulator-fungsi](https://kalkulator-fungsi.vercel.app)

## Fungsi Komposisi 

Fungsi komposisi adalah gabungan dari dua fungsi atau lebih sehingga menghasilkan suatu fungsi baru.

## Fungsi Invers

Fungsi invers adalah fungsi yang "membalikkan" efek dari fungsi asli. Jika kita memiliki sebuah fungsi f yang memetakan elemen dari himpunan A ke himpunan B, maka fungsi invers f⁻¹ memetakan elemen dari himpunan B kembali ke himpunan A
